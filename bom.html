BOM assignment

Upload or paste a BOM, review candidate LCSC parts, and export a CSV for JLCPCB.

Import BOM

Supports CSV, TSV, and copy-pasted table data. Common columns such as designator, quantity, value/comment, package/footprint, MPN, and LCSC part number are detected automatically.

Export format

The exported CSV contains the columns typically required for JLC assembly uploads.

CommentValue or description from the BOM row
DesignatorReference designators exactly as uploaded
FootprintFootprint or package column from the BOM row
LCSC Part #Assigned LCSC part number
