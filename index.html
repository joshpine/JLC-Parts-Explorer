01 / SEMANTIC PARSING

Shorthand parsing

Equivalent values and package aliases are normalized during search. For example, 100n, 0.1uF, and 100000pF resolve to the same value, and metric package aliases are mapped to the same footprint family.

100n 0402 x7r → Cat:Cap Val:100nF Pkg:0402 TempCo:X7R
02 / TIERED RESULTS

Grouped results

Search results are grouped into JLC Basic, extended parts from larger global manufacturers, and extended parts from other manufacturers so you can compare sourcing options quickly.

Parametric filters

Parts can be filtered by package, tolerance, voltage, temperature coefficient, brand, and stock state. Parsed numeric values are normalized so equivalent queries map to the same filter set.
